데이터가 중요!
----------
1) 기본형 데이터- 정수, 실수, 문자, 논리,변수에 "값"이 들어감.
   언어의 기본기능으로 여러가지 처리를 할 수 있게 만들어놓았음.
   int x = 100;
   int y = 200;
   x == y
2) 참조형 데이터- 나머지 다, 배열, 컬렉션, 클래스, 인터페이스 ~~
   변수에 주소가 들어감.
   String x = "ok";
   String y = "no";
   x == y(에러)
   x.equals(y)
   
   
   
   1) 두 수를 입력받아서
      동일하면 동일해요!
      동일하지 않으면 달라요! 출력
   2) id가 root라고 입력하면 맞아요!
      id가 root가 아닌 다른 것으로 입력하면 틀려요! 출력
   3) id가 root, pw가 1234 입력이 둘다 맞으면 로그인 성공!
      둘 중 하나라도 틀리면 로그인 실패!
